EDHRec Clipboard

EDHRec is a statistics site for the most popular Magic the Gathering format. The clipboard is a feature that bridges the gap between EDHRec and deckbuilding or card shopping sites. The feature allows users to build a list of cards, then either export that list or pre-poulate a cart on the most popular shopping sites.

Messenger MTG Card Fetcher

MTGCardFetcher is a messenger bot I created in order to more easily find and share magic cards on mobile. The bot uses Facebook Messenger's send api along with Scryfall's card api to serve shareable pictures to users. To see the bot in action, message MTGCardFetcher on Facebook Messenger.
